About MoneyMinds

MoneyMinds is an easy-to-access website where adolescents can begin to learn and engage in finance. There are three main tools provided in this website:

    1. Calculator: Either the 'Savings Calculator' which calculates how much the user will save in a period of time, or the 'Goal Calculator' which calculates how much the user will have the save per month to achieve a goal. The 'Savings Calculator' also has the option for the user to input the annual interest rate of the preferred bank if the user wishes to save the money in a bank. Therefore, the calculator will then calculate how much the user will save in a period of time taking the annual interest rate of the bank into account.
